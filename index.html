An interactive course in 12 acts

Under the Hood

You already operate AI agents in production. This is the journey to the other side of the glass: read any Python, YAML, or agent-framework code fluently — then build the machinery yourself.

How this works

Every chapter opens with a story from the real world, builds the idea in plain language, then shows you the actual code — line by line, with every line explained. You run Python right in the browser, step through animated diagrams, and check yourself with a short quiz. Mark chapters complete as you go; your progress is saved on this device.

Read in order for the full journey, or jump straight to what tomorrow's meeting needs. Either way: nothing here assumes you have ever written code by hand — and nothing here talks down to you.

Three promises

  • Fluency, not memorisation. The goal of every chapter is that a page of real code stops looking like a foreign language. The five "Reading real code" chapters are checkpoints where you prove it to yourself.
  • The full stack, honestly. Python, models and PKL files, transformers, LLM APIs, agents from scratch, LangChain, LangGraph, deep agents, MCP and A2A, harness anatomy, the 2026 landscape, and the GitLab workflow you'll ship through — including where each tool is weak.
  • Joy over grind. Stories, animations, runnable code, flip cards. If a chapter ever feels like a textbook, that chapter has failed.
